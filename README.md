# 第一題 Python實現單項鏈表的最全教程
## 概述
單向鏈結串列（英語：singly linked list）是鏈結串列的一種，其特點是鏈結串列的鏈結方向是單向的，對鏈結串列的存取要通過從頭部開始，依序往下讀取。

一個單向鏈結串列的節點被分成兩個部分。第一個部分儲存或者顯示關於節點的資訊，第二個部分儲存下一個節點的位址。單向鏈結串列只可向一個方向節點走訪。
